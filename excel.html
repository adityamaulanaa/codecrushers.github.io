Microsoft Excel



Sejarah


Microsoft Excel merupakan perangkat lunak pengolah angka (spreadsheet) yang dikembangkan oleh Microsoft. Pertama kali dirilis pada tahun 1985 untuk sistem operasi Macintosh, Excel kemudian hadir untuk Windows pada tahun 1987 dengan Excel versi 2.0. Excel menggantikan perangkat lunak spreadsheet sebelumnya seperti Lotus 1-2-3 yang populer pada masa itu. Seiring waktu, Microsoft Excel menjadi bagian dari Microsoft Office dan terus mengalami perkembangan fitur dan performa hingga versi terbaru saat ini.




Definisi


Merupakan perangkat lunak pengolah angka berupa spreadsheet. Excel digunakan untuk bekerja dengan daftar data, menghitung angka-angka, membuat laporan, diagram, grafik, dan sebagainya. Beberapa Materi yang sudah diajarkan di Microsoft Excel diantaranya :

  • Fungsi Aritmatika
  • Fungsi COUNT
  • Fungsi IF Tunggal dan Bertingkat
  • Fungsi IF AND dan IF OR

    • Selain itu, Microsoft Excel juga memiliki fungsi utama sebagai berikut :
  • Menghitung Angka Secara Otomatis menggunakan rumus.
  • Membuat Grafik dari data yang ada.
  • Analisis Data dengan fitur seperti PivotTable dan Filter.



    • Beberapa Rumus Penting pada Microsoft Excel


    Rumus Fungsi
    =SUM(Sel awal : Sel Akhir) Menjumlahkan nilai pada sel
    =AVERAGE(Sel Awal : Sel Akhir) Menghitung nilai rata rata pada sel
    =MIN(Sel Awal : Sel Akhir) Mencari nilai terkecil pada sel
    =MAX(Sel Awal : Sel Akhir) Mencari nilai terbesar pada sel
    =RANK(Sel Awal : Sel Akhir) Mencari peringkat melalui perbandingan nilai pada masing-masing sel
    =COUNT(Sel Awal : Sel Akhir) Menghitung jumlah sel yang terisi data
    =COUNTBLANK(Sel Awal : Sel Akhir) Menghitung jumlah sel yang tidak terisi data
